📅 TinyMonth
A modern, intuitive calendar management application built with Next.js and React for tracking time, managing schedules, and organizing your life with color-coded calendars and smart features.

✨ Key Features:

  • 🗓️ Smart Calendar Management with color-coded organization
  • 📊 Comprehensive Statistics and annual overviews
  • 💬 Comment System for adding notes to any day
  • 🔄 Import/Export functionality for data backup
  • 🎨 Beautiful, responsive interface with Tailwind CSS
